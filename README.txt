This program provides a graphical user interface (GUI) for the DS Messenger application. 
It allows users to send and receive messages through the DS Messenger service.

This program is a simple messaging application with a basic interface. 
It lets you send and receive messages easily. You can add new contacts, configure your messaging server, and check for new messages. 
Just run the a5.py file to start the program.
